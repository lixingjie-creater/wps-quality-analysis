\# 焊接工艺(WPS)质量监控数据分析



\## 项目背景

基于某工程项目焊接工序的真实生产数据（约15万条记录），开发自动化报表工具，用于质量监控和异常预警。



\## 数据说明

\- 原始数据包含管线号、焊口号、WPS编号、材质、焊接日期、NDT检测结果等字段。

\- 因数据量较大，本仓库仅提供前50行示例结构（前50行数据.xlsx）。



\## 分析内容

\- 数据清洗：字段智能匹配、状态归一化（合格/不合格/检测中）

\- 多维聚合：管线质量汇总、焊工绩效、WPS-材质交叉表

\- 趋势分析：月度焊接进度与RT合格率变化

\- 异常标记：自动标红RT合格率低于80%的WPS和焊工



\## 输出结果

\- 李星杰-运行结果明细.xlsx 包含9张工作表及趋势图

\- 一键生成，无需手动整理



\## 如何使用

1\. 安装依赖：`pip install pandas numpy xlsxwriter`

2\. 将你的完整数据文件放入 目录，修改代码中的路径

3\. 运行 `李星杰-测试题代码.ipynb`
